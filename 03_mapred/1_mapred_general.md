# What is MapReduce?

!SLIDE bullets incremental
# MapReduce is NOT...

* a query language
* a framework
* a database
* just Hadoop

!SLIDE bullets incremental
# MapReduce is a computational style.

* Two functions
* **Map** : single items
* **Reduce** : collections

!SLIDE bullets incremental
# Why MapReduce?

* Easy distributed processing
* Data locality
* Scales up and down
* "It's just code."

!SLIDE bullets incremental
# Understanding Map

* 1 datum → 1 call
* Extract
* Transform
* Filter

!SLIDE bullets incremental
# Understanding Reduce

* List of N data → M calls
* Sum, Count, Group
* Sort, Limit
* Minimum, Maximum, Stats
